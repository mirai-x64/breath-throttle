# 息継ぎのコーナー — ブレーキは黙ることだけ

声を出している間だけアクセルが踏まれる、トップダウンのタイムアタック。
**声量がそのままスロットル開度**になり、**減速手段は黙ることしかない。**

縁日の帰り道、机の上に積み木と消しゴムで作ったコースを、紙の風船が走る。

単一 HTML + Canvas 2D。フレームワークもビルド工程も無い。`index.html` を開くだけで動く。

## 遊びかた

1. マイクを許可する
2. キャリブレーション ── 3秒黙って「床」、3秒声を出して「天井」を測る
3. 直線で吐ききり、コーナーの手前で黙って息を吸う

操舵は `←` `→` または `A` `D`。`R` でやり直し。

## 中身

### ブレーキが無いということ

アクセルは開度 `t`(声量)、推力は `A·t^γ`。抗力は速度比例 + 一定摩擦。

```
v' = A·t^2.4 − 0.7·v − 60
```

黙れば `t=0` になり、惰性と抗力だけが残る。**減速入力は存在しない。**
高速から曲がれる速度まで落とすのに約 0.5 秒かかる ── その 0.5 秒を
どこで作るかがこのゲームの全部。

### 速度が開度に対して非線形であること

`γ = 2.4` と一定摩擦 `C = 60` の組み合わせで、終端速度はこうなる。

| 開度 | 0.5 | 0.6 | 0.7 | 0.8 | 0.9 | 1.0 |
|------|-----|-----|-----|-----|-----|-----|
| 終端速度 (px/s) | 10 | 64 | 127 | 205 | 299 | 420 |

開度 0.48 未満では推力が摩擦に負けて**まったく動かない**(HUD の白線がその境界)。
小声のハミングで流し続ける戦法は、ここで潰れている。

### 曲がれる速度に上限があること

横加速度に上限 `a = 520 px/s²` があるので、旋回半径は `v²/a` で速度の二乗に比例する。
コーナー半径 62〜78px のコースでは上限が 224〜247 px/s ── 最高速 420 の約半分。
**直線で出した速度のままではコーナーに入れない。**

### 壁

当たり判定は中心線からの距離(描画の帯と厳密に一致する)。触れると
突っ込み角に比例して速度を失い、さらに接触中は速度が 110 px/s で頭打ちになる。
壁を「無料のブレーキ兼ステアリング」として使う戦法を潰すための上限。

## 「黙る方が速い」ことの検証

体感ではなく実測で確かめてある。`window.__sim()` が、実走と同じ物理・
同じコース・同じ操舵AI(反応遅れ 0.14 秒つき)で、**スロットル方針だけを
差し替えて**周回タイムを比較する。ブラウザのコンソールで実行できる。

実ゲームループ(実際の `requestAnimationFrame`・実際のキーハンドラ)での結果:

| 走りかた | 1周タイム |
|---|---|
| 小声で流し続ける (開度 0.6) | **63.66 秒** |
| 直線で全開 / コーナーで黙る | **14.42 秒** |

4.4 倍の差がある。さらに厳しい比較として、`__sim()` で一定開度を
0.50〜1.00 まで掃引した最良値(壁に当たりながら全開で走り続ける戦法を含む)
と比べても、変調したほうが **12% 速い**。

## マイクが使えないとき

スペースキーにフォールバックするが、**動作確認用であり遊びではない**旨を
画面に出し続ける。押しっぱなしにできるキーでは、このゲームの中身
(息が続かないこと)が再現されないため。

## 作らなかったもの

紙風船は走行中しぼんでいくが、**見た目だけ**で、時間経過で不利になる
ルールは入れていない。ひねりは「声がアクセル」の一つに絞ってある。
